01

O efeito
patina
como gelo.

A interpolação linear (lerp) entre a posição atual e o destino cria uma desaceleração exponencial — quanto mais próximo do alvo, mais lento fica. O resultado é a sensação de inércia, exatamente como uma patinação no gelo.

02

Fator lerp
controla
a inércia.

Com t = 0.07, a posição atual percorre 7% da distância restante a cada frame. A 60fps isso cria um deslize natural de ~350ms. Reduza para 0.04 para gelo, aumente para 0.15 para um momentum mais leve.

Como configurar

01

HTML / CSS

overflow: hidden no html/body. Container fixo com will-change: transform. Proxy invisible com a altura real do conteúdo.

02

JavaScript

Escuta window.scroll para obter o target. Loop com requestAnimationFrame aplicando lerp a cada frame.

03

Fator Lerp

0.04 gelo extremo · 0.07 patinação ideal · 0.12 suave moderado · 0.25 momentum leve
